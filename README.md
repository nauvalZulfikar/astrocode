# AstroCode

Game education berbasis Godot 4 — kombinasi astronomi + coding.

## Stack

- **Engine:** Godot 4 (`project.godot`)
- **Main scene:** `scenes/world/main_world.tscn`

## Layout

```
astrocode/
├── addons/        Godot plugins
├── assets/        Sprites, audio, fonts
├── docs/          Design docs (GDD, level design, dst)
├── resources/     Game resources (.tres)
├── scenes/        Scene tree (.tscn)
├── scripts/       GDScript
└── project.godot
```

## Run

Buka folder di **Godot 4 editor** → F5 (play).

## Lihat juga

- `docs/` — GDD, art direction, curriculum, level design
