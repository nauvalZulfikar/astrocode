# Character Design References

Visual references for each playable/NPC character. Pairs with the asset briefs in `docs/briefs/`.

Use for: handing to pixel artists, animators, marketing, and Steam page assets. NOT consumed by the game runtime — actual in-game sprites live in `assets/sprites/`.

## Structure

```
docs/character-design/
├── README.md          ← this file
└── kade/              ← protagonist (astronaut, Thyra-7)
    └── kade-rotation-preview.gif   ← PixelLab 8-dir cycle preview
```

Add a subfolder per character as references arrive (Mira, Cass, Sable, Wren, Rootknot, etc.).
